Setup Instructions

⚠️ Important: Your GitHub token was detected in the code and blocked for security. This is a security feature. Follow these instructions to set up the token securely.

Option 1: Use GitHub Repository Secret (Recommended)

  1. Go to your repository settings: Settings → Secrets → Actions
  2. Click "New repository secret"
  3. Name: RESEARCH_TRIGGER_TOKEN
  4. Value: Your GitHub personal access token
  5. Click "Add secret"
  6. Update the workflow to use this secret

How to Create a GitHub Token

  1. Go to GitHub Token Settings
  2. Name: "Daily Top Gainers Research"
  3. Expiration: 90 days or more
  4. Select scopes: ✓ repo and ✓ workflow
  5. Click "Generate token"
  6. Copy the token (starts with ghp_)
